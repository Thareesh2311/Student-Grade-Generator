Project Overview
Student Grade Generator is a Java Swing GUI application that helps students analyze their academic
performance.
The application takes marks of 5 subjects and automatically calculates: Total Marks Average Marks
Highest Marks Lowest Marks Final Grade Technologies Used Java Java Swing AWT Event Handling
Features User-Friendly GUI Automatic Total Calculation Average Calculation Highest & Lowest Marks
Detection Grade Generation Clear & Exit Buttons Grade System
90+ → A+
80+ → A
70+ → B
60+ → C
50+ → D
Below 50 → Fail

How to Run
Compile:
javac StudentGradeGeneratorGUI.java
Run:
java StudentGradeGeneratorGUI
